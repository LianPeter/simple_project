# Monorepo 项目说明

本仓库为多项目管理的 Monorepo，统一管理前端与后端代码，提升开发效率与协作体验。

## 目录结构

```
/packages      # 各独立模块或应用
/scripts       # 项目脚本工具
/docs          # 项目文档
/apps         # 前端与后端应用
  /frontend    # 前端应用
  /backend     # 后端应用
README.md      # 项目说明
```

## 快速开始

1. 安装依赖

   ```bash
   pnpm install
   ```

2. 启动项目

   ```bash
   pnpm run dev
   ```

## 贡献指南

1. Fork 本仓库
2. 新建分支进行开发
3. 提交 Pull Request
